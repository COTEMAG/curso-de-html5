Curso Básico de HTML5

1. ¿Qué es HTML5?

HTML5 es el lenguaje estándar para crear páginas web. Consiste en etiquetas (tags) que definen la estructura y contenido de una página.

Ejemplo básico:

<!DOCTYPE html>
<html>
<head>
    <title>Mi página</title>
</head>
<body>
    <h1>Hola Mundo</h1>
</body>
</html>

2. ¿Qué son etiquetas, atributos y valores?

Etiquetas (Tags)

Son palabras clave entre < > que definen elementos HTML. Pueden ser:

  • De apertura: <etiqueta>
  • De cierre: </etiqueta> (algunas no lo necesitan)

Atributos

Son modificadores que van dentro de la etiqueta de apertura y proporcionan información adicional:

<etiqueta atributo="valor">Contenido</etiqueta>

Atributos globales comunes

Algunos atributos globales podrian ser id y tambien class, son de los mas utilizados ya que permiten darle una clase o un id especifico a cada elemento HTML.

El atributo id es para darle un identificador unico al elemento, mientras que el atributo class agrupa elementos para aplicar estilos o comportamientos a ciertos elementos

Ejemplo de clase

    <h1 class="titulo-especial"> Titulo unico <h1>

Ejemplo de identificador (id)

    <p id="parrafo-importante"> Este es un parrafo importante <p>

Se recomienda usar las clases para varios elementos, mientras que las ID solo para un unico elemento en todo el archivo HTML.

Valores

Es la configuración específica del atributo, siempre entre comillas:

<img src="foto.jpg" alt="Mi imagen">

3. Etiquetas básicas

Encabezados (h1 a h6)

<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Sub-sección</h3>

Título principal

Subtítulo

Sub-sección

Párrafos (p)

<p>Este es un párrafo.</p>
<p>Este es otro párrafo.</p>

Enlaces (a)

Usa el atributo href para especificar el destino:

<a href="https://www.google.com">Ir a Google</a>

Importancia (strong)

La etiqueta strong se utiliza para definir texto de gran importancia. El contenido que contiene se muestra en negrita:

<strong>Este texto es importante<strong>

Negrita (b)

La etiqueta b especifica texto en negrita sin ninguna importancia adicional:

<b>Este texto está en negrita</b>

Imágenes (img)

Etiqueta auto-cerrada con dos atributos obligatorios:

<img src="ruta/de/imagen.jpg" alt="Texto alternativo">

4. Listas y tablas

Lista no ordenada (ul)

<ul>
    <li>Manzana</li>
    <li>Banana</li>
</ul>

Lista ordenada (ol)

<ol>
    <li>Primer paso</li>
    <li>Segundo paso</li>
</ol>

Tablas (table, tr, th, td)

<table border="1">
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>Ana</td>
        <td>25</td>
    </tr>
</table>

5. Formularios (form)

Para recolectar datos del usuario:

<form>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre"><br>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br>

    <input type="submit" value="Enviar">
</form>

6. HTML5 Semántico

Etiquetas que mejoran la organización y accesibilidad:

<header>Encabezado</header>
<nav>Menú de navegación</nav>
<main>
    <article>Contenido principal</article>
    <aside>Contenido secundario</aside>
</main>
<footer>Pie de página</footer>

7. Metaetiquetas

Las metaetiquetas son etiquetas HTML que proporcionan información sobre una pagina web al cliente (motores de búsqueda/navegador), son datos no-visibles pero que ayudan al cliente (navegador) a entender de que trata la pagina y a mostrarla correctamente en los resultados de búsqueda.

Las etiquetas <meta> siempre van dentro del elemento <head> y se utiliza normalmente para especificar la descripción de la página, las palabras clave, el autor del documento, entre otros.

Las etiquetas <meta> no serán visibles en la página, pero el cliente (navegador) lo analizará para entenderla mejor. Estas etiquetas se usan semánticamente para lograr una mejor posición de tu página web.

Para definir palabras clave

Se utiliza el name para especificar el nombre del metadato y luego se usa content para especificar el valor asociado con el nombre del atributo, ejemplo 

  <meta name="keywords" content="HTML, css, javascript">

Siempre se deben separar por una coma (,) las palabras clave.

Para definir al autor

  <meta name="author" content="Nombre">

Para ponerle descripcion a tu página web

  <meta name="description" content="descripcion de la página">

En la descripción de la página se recomienda no usar más de 150 carácteres

8. Página web completa

<!DOCTYPE html>
<html>
<head>
<title>Mi primera página</title>
<meta name="author" content="John Doe">
<meta name="description" content="este es un ejemplo de descripcion">
<meta name="keywords" content="html, css, web, primera pagina">
</head>
<body>
    <header>
        <h1>Bienvenido</h1>
    </header>
    <nav>
        <a href="#inicio">Inicio</a> |
        <a href="#contacto">Contacto</a>
    </nav>
    <main>
        <section>
            <h2>Sobre mí</h2>
            <p>¡Hola! Esta es mi primera página.</p>
            <img src="foto.jpg" alt="Mi foto" width="200">
        </section>
    </main>
    <footer>
        <p>© 2023 - Todos los derechos reservados</p>
    </footer>
</body>
</html>

La etiqueta <thead> define la parte superior de la tabla (encabezado), donde se suelen colocar los títulos de cada columna usando <th>. El contenido principal va en <tbody>, con las celdas de datos normales escritas como <td>.
