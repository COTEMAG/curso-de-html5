Curso Básico de HTML5

1. ¿Qué es HTML5?

HTML5 es el lenguaje estándar para crear páginas web. Consiste en etiquetas (tags) que definen la estructura y contenido de una página.

Ejemplo básico:

<!DOCTYPE html>
<html>
<head>
    <title>Mi página</title>
</head>
<body>
    <h1>Hola Mundo</h1>
</body>
</html>

2. ¿Qué son etiquetas, atributos y valores?

Etiquetas (Tags)

Son palabras clave entre < > que definen elementos HTML. Pueden ser:

  • De apertura: <etiqueta>
  • De cierre: </etiqueta> (algunas no lo necesitan)

Atributos

Son modificadores que van dentro de la etiqueta de apertura y proporcionan información adicional:

<etiqueta atributo="valor">Contenido</etiqueta>

Valores

Es la configuración específica del atributo, siempre entre comillas:

<img src="foto.jpg" alt="Mi imagen">

3. Etiquetas básicas

Encabezados (h1 a h6)

<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Sub-sección</h3>

Título principal

Subtítulo

Sub-sección

Párrafos (p)

<p>Este es un párrafo.</p>
<p>Este es otro párrafo.</p>

Enlaces (a)

Usa el atributo href para especificar el destino:

<a href="https://www.google.com">Ir a Google</a>

Imágenes (img)

Etiqueta auto-cerrada con dos atributos obligatorios:

<img src="ruta/de/imagen.jpg" alt="Texto alternativo">

4. Listas y tablas

Lista no ordenada (ul)

<ul>
    <li>Manzana</li>
    <li>Banana</li>
</ul>

Lista ordenada (ol)

<ol>
    <li>Primer paso</li>
    <li>Segundo paso</li>
</ol>

Tablas (table, tr, th, td)

<table border="1">
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>Ana</td>
        <td>25</td>
    </tr>
</table>

5. Formularios (form)

Para recolectar datos del usuario:

<form>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre"><br>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br>

    <input type="submit" value="Enviar">
</form>

6. HTML5 Semántico

Etiquetas que mejoran la organización y accesibilidad:

<header>Encabezado</header>
<nav>Menú de navegación</nav>
<main>
    <article>Contenido principal</article>
    <aside>Contenido secundario</aside>
</main>
<footer>Pie de página</footer>

7. Página web completa

<!DOCTYPE html>
<html>
<head>
    <title>Mi primera página</title>
</head>
<body>
    <header>
        <h1>Bienvenido</h1>
    </header>
    <nav>
        <a href="#inicio">Inicio</a> |
        <a href="#contacto">Contacto</a>
    </nav>
    <main>
        <section>
            <h2>Sobre mí</h2>
            <p>¡Hola! Esta es mi primera página.</p>
            <img src="foto.jpg" alt="Mi foto" width="200">
        </section>
    </main>
    <footer>
        <p>© 2023 - Todos los derechos reservados</p>
    </footer>
</body>
</html>
